Cómo funciona
1. Sube el archivo CSV

El archivo debe tener el formato estándar de John Deere con datos de utilización.

2. Extracción automática

El sistema extraerá automáticamente el PIN del equipo del nombre del archivo.

3. Creación automática de equipo

Si el equipo no existe, se creará automáticamente con datos básicos.

4. Análisis automático

El sistema procesará los datos y generará recomendaciones automáticas.

5. Genera el reporte

Obtén un PDF profesional con análisis y recomendaciones para el cliente.

Ejemplo de datos

El CSV debe contener datos como:

  • Utilización del motor (Carga alta, mediana, baja)
  • MFWD Utilization per gear
  • Modos de funcionamiento
  • Tiempo en marcha
  • Nivel del depósito de combustible
